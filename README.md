## 📌 Project Overview

**IronForge** is a premium, fully responsive fitness website designed to showcase fitness programs, trainers, pricing plans, and member testimonials. Built with a modern dark theme and vibrant orange/purple accents, this project demonstrates advanced HTML5 and CSS3 techniques including CSS Grid, Flexbox, and responsive media queries.

> **🏢 Internship Project at Decode Labs**

---

## 🎯 Features

| Feature | Description |
|---------|-------------|
| ✅ **Fully Responsive** | Works flawlessly on desktop, tablet, and mobile |
| ✅ **Dark Theme** | Premium modern aesthetic with purple/orange accents |
| ✅ **CSS Media Queries** | 4 responsive breakpoints for optimal viewing |
| ✅ **Responsive Navigation** | Hamburger menu on mobile devices |
| ✅ **CSS Grid** | Macro page layouts (programs, trainers, pricing) |
| ✅ **CSS Flexbox** | Micro component alignment (navigation, buttons) |
| ✅ **HD Images** | Real fitness models and athletes (no emojis) |
| ✅ **No JavaScript** | Except mobile menu toggle + smooth scroll |
| ✅ **Smooth Scroll** | Anchor link navigation with smooth scrolling |

## 🎨 Design System

### Color Palette

| Color | Hex Code | Usage |
|-------|----------|-------|
| Primary Orange | `#ff6b35` | Buttons, accents, highlights |
| Secondary Purple | `#7c3aed` | Secondary accents |
| Dark Background | `#0a0a0f` | Body background |
| Card Background | `#1a1a24` | Cards, sections |
| Text Light | `#ffffff` | Headings |
| Text Muted | `#8a8aa0` | Body text |

### Typography
- **Font Family:** System UI / Segoe UI / Roboto
- **Headings:** Bold, clean, and modern
- **Body:** Readable, well-spaced, professional

## 📱 Responsive Breakpoints

| Device | Screen Width | Layout |
|--------|--------------|--------|
| **Desktop** | 1024px+ | Full 4-column layout |
| **Tablet** | 768px - 1024px | 2-column layout |
| **Mobile** | 480px - 768px | Single column, hamburger menu |
| **Small Mobile** | Below 480px | Compact layout |
